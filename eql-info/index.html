1) EverQuest Legends Overview Core

EverQuest Legends (EQL) is a classic-inspired EverQuest experience with modernized systems and quality-of-life updates.

Focus areas

  • Classic world framing and progression-focused gameplay.
  • Solo-friendly and small-group friendly progression options.
  • Build diversity through expanded class combination mechanics.

Development

  • EverQuest IP operations remain under Daybreak / Darkpaw.
  • EQL development is a collaboration with Game Jawn.

2) Classes and Class Combinations Builds

Class pool

EQL draws from core EverQuest archetypes — tank, healer, caster, melee DPS, and support — spanning classic and expansion-era classes.

  • Warrior, Paladin, Shadow Knight, Cleric, Shaman, Druid, Ranger, Bard.
  • Wizard, Magician, Enchanter, Necromancer, Rogue, Monk, Berserker, Beastlord.

Combination model

  • Trio/multiclass structure: primary + secondary + tertiary classes.
  • 560+ potential trio combinations depending on restrictions and version.
  • Race-based restrictions apply to primary class selection; more flexibility for secondary/tertiary.
